Sobre o Sistema

Projeto ativo
SoftBot Pro WhatsApp

Sistema de automação de atendimento com painel administrativo

O SoftBot Pro é um projeto desenvolvido para simular e preparar uma automação de atendimento via WhatsApp Business, com API em Python, painel web, controle de usuários, FAQs, mensagens, relatórios e estrutura preparada para integração real.

Objetivo

Criar uma base profissional para atendimento automatizado, permitindo testar mensagens, cadastrar respostas frequentes, acompanhar históricos e preparar uma futura integração com a API oficial do WhatsApp.

Status atual

O sistema está funcionando em ambiente local, com autenticação, painel administrativo, gerenciamento de usuários, relatórios e simulação de mensagens.

Uso principal

Projeto criado para estudo, portfólio e prática de desenvolvimento back-end, front-end, APIs, autenticação, banco de dados e automação.

Tecnologias utilizadas

Principais ferramentas usadas no desenvolvimento.

Python FastAPI Uvicorn SQLite SQLAlchemy Pydantic JWT HTML CSS JavaScript WhatsApp Cloud API Git

Funcionalidades principais

O que o sistema já consegue fazer.

Autenticação

Login com token JWT e proteção de páginas do painel.

Usuários e permissões

Admin, atendente e suporte, com controle de acesso por perfil.

Mensagens

Registro de mensagens, alteração de status, busca e filtros.

FAQs

Cadastro de perguntas frequentes para respostas automáticas.

Simulador

Envio de mensagens de teste para validar o comportamento do bot.

WhatsApp

Estrutura preparada para webhook e envio via WhatsApp Business Platform.

Relatórios

Resumo de atendimentos, status das mensagens, usuários e taxa de resolução.

Configurações

Página de diagnóstico com status da API, banco, ambiente e integração.
